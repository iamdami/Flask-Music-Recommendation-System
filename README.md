# Flask Wep App
A music recommendation system that recommends some songs based on user mood trained by using [FER2013 dataset](https://paperswithcode.com/dataset/fer2013).  
<br>

## Tech Stack
- Python 3.9.5  
- Flask 2.0.2  
- Opencv 4.5.5  
- Tensorflow 2.8.0  
- VsCode 1.52.1
